Abstract

This paper introduces Zero-Knowledge Multi-Robot Task Allocation (ZK-MRTA), a new problem formulation for decentralized task allocation in which agents have no prior knowledge of task attributes, no knowledge of their own or others' capabilities, and no direct communication channel. Under these constraints, agents learn exclusively from the outcomes of their own actions and from indirect public evidence generated by the swarm — a strictly harder setting than classical MRTA, which typically assumes that costs, feasibility, or compatibility are known or communicable. By elevating these informational restrictions from implementation challenges to defining assumptions, ZK-MRTA establishes a distinct class of task-allocation problems with direct relevance to contested, communication-denied, and open-world deployment scenarios.

To address this setting, this paper proposes a decentralized collaborative-filtering-inspired approach based on matrix factorization. The key reinterpretation is that collaborative filtering is used not for preference prediction, but for latent utility estimation: agents infer hidden task suitability from sparse interaction outcomes accumulated over time. Each agent maintains a private local latent model and independently selects tasks using estimated utilities, while coordination emerges indirectly through shared public observations rather than explicit parameter exchange.

To support rigorous evaluation, the paper presents a simulation benchmark for ZK-MRTA implemented as a Gymnasium-compatible PettingZoo parallel multi-agent environment. The framework enforces zero-knowledge observability constraints, generates controllable hidden compatibility structures, and provides policy-agnostic performance metrics including task completion efficiency, overkill, collision rate, and latent match quality. The main contribution of this work is threefold: the formalization of the ZK-MRTA problem and its associated performance metrics, the introduction of a family of decentralized collaborative-filtering-based policies that learn latent utility under strict informational constraints, and the development of a modular benchmark for systematic empirical analysis of task-allocation strategies in hidden, communication-free environments.
